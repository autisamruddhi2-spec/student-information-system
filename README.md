# Student Information System (Java)

## Project Overview
This is a menu-driven Java console application developed to manage student records.
It allows users to add, view, update, delete, and search student information.

## Features
- Add Student
- View Students
- Update Student
- Delete Student
- Search Student by ID or Name
- Input validation for age

## Project Version
Standard Version (Add, Search, Update functionality)

## Technologies Used
- Java
- OOP Concepts (Encapsulation)
- ArrayList
- Scanner

## Setup Instructions
1. Install Java JDK 8 or above
2. Clone the repository
3. Navigate to `src/` directory
4. Compile:
   javac Student.java StudentInformationSystem.java
5. Run:
   java StudentInformationSystem

## Code Structure
- Student.java ? Student class with encapsulation
- StudentInformationSystem.java ? Menu system and logic
